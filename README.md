# markdown2image

> transform markdown to img

## Installation

```shell
# install it globally
$ npm install -g markdown2image

# or yarn
$ yarn global add markdown2image
```

## Usage

```shell
$ markdown2image <input> [-o|--output] [-w|--width]
Create img complete.
```
